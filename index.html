How Do I Use Super?

In Python 2.7:

super(MyClass, self).my_method(parameters) # with new-style class

MyClass.my_method(self, parameters) # with old-style class

In Python 3:

super().my_method(parameters)

In CoffeeScript:

super parameters # Super-method is inferred

In ECMAScript 6:

super(parameters) // In constructor

super.myMethod(parameters) // Normal methods

In Objective-C:

[super myMethod:parameters]

In Ruby:

super # Super-method and parameters are inferred

super(parameters) # Super-method is inferred

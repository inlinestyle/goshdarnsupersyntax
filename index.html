How Do I Use Super?

In Python 2.7 (new-style classes):

super(MyClass, self).my_method(parameters)

In Python 3:

super().my_method(parameters)

In CoffeeScript:

super parameters

In ECMAScript 6:

super(parameters) // In constructor super.my_method(parameters) // Normal methods
This site is not intended to be an exhaustive list of all possible uses of super.
